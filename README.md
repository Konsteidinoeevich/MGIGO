This is a trajectory planning project using receding horizon style(or MPC, if you are used to call it) using global optimization method based on IGO and GMM as searching distribution. 
These codes are just for trials as numerical examples that try tu verify whether a cost function is correct. Also, the codes included in GMM_IGO are solvers equipped with different types.

If you want to figure out the algorithm in the article MGIGO-MPC you can just use these codes. For practicle engineering uses, see my friend's projects. He turned some examples and the solver into CUDA version.
